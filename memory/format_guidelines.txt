# 医学論文読解問題 形式ガイドライン

## 概要
このガイドラインは、医学論文読解問題の形式と構造に関する要件を定義します。一貫性のある問題形式を維持するための基準となります。

## 既存例題の確認
新規例題作成前に、english_questionディレクトリ内の既存例題を必ず確認し、以下の点に注意してください：

1. 重複の防止
   - 既存例題で扱われている疾患領域の確認
   - 使用されている研究手法や治療アプローチの確認
   - 診断技術や治療戦略の重複を避ける
   - 分子メカニズムの重複を避ける

2. 新規例題作成の方向性
   - 未カバーの医学分野の選択
   - 異なる研究アプローチの採用
   - 新しい技術や方法論の導入
   - 独自の臨床的意義の提示

## ファイル構造要件

### 1. 問題ファイル（例題_{番号}_問題.md）
```
# Example Question {N}: [タイトル] ([単語数] words)
# [日本語タイトル]（[単語数]語）

## Original Text / 英文
[段落番号付きの英文]

## Questions / 設問
1. [第1問]
2. [第2問]
3. [第3問]
```

### 2. 解答・解説ファイル（例題_{番号}_解答解説.md）
```
# [問題ファイルと同じタイトル]

## Original Text with Translation / 原文と和訳
[段落番号付きの英文と和訳]

## Answer Guide / 解答解説

### 問2の解答例

```

## 形式要件

1. 文章の形式
   - 長さ：600-800語
   - 段落番号の付与
   - 適切なマークダウン形式の使用
   - 両言語での単語数の明記

2. 設問形式
   - 3問構成
   - 各設問は明確な指示を含む
   - 設問番号の一貫した表記

3. 解答形式
   - 箇条書きによる模範解答
   - 各設問の解答は独立したセクション
   - 統一された評価基準の提示

4. 言語サポート形式
   - 語彙項目：8-10個
   - 文法ポイント：3-4個
   - 読解ストラテジーの提示
